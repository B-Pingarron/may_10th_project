# May 10th Project - Conflict Death Toll Data Study

## Overview
A comprehensive data study analyzing death toll figures across current world conflicts, with Francesca Albanese's UN reports as the foundational pillar.

## Primary Focus Areas
- Palestine, Gaza, West Bank
- Lebanon
- Sudan
- Other current world conflicts

## Key Research Pillar
Francesca Albanese - UN Special Rapporteur on the situation of human rights in the Palestinian territories occupied since 1967

## Project Goals
1. Gather authoritative death toll data from multiple conflicts
2. Cross-reference with Francesca Albanese's reports
3. Create comparative analysis of conflict casualties
4. Present findings in a comprehensive data study
